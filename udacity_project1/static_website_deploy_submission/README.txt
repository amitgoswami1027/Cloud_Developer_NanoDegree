Udacity project help to deploy the static website on AWS.

The files included are: 
index.html - The Index document for the website.
/img - The background image file for the website.
/vendor - Bootssrap CSS framework, Font, and JavaScript libraries needed for the website to function.
/css - CSS files for the website.

STEPS:
1. Created the S3 bucket with the name “udacity-project1-deploy-static-website”.
2. Upload Project files into the S3 bucket.
3. Secure the S3 bucket via IAM
4. Configure S3 Bucket
5. Distribute website via cloud front
6. Travel Blog Website (Website display in Browser)
7. Customization
